# Chromadb Assistant

### 1. Create a virtual environment

```shell
python3 -m venv ~/.venvs/aienv
source ~/.venvs/aienv/bin/activate
```

### 2. Install libraries

```shell
pip install -U chromadb pypdf openai phidata
```

### 3. Run Assistant

```shell
python cookbook/integrations/chromadb/assistant.py
```
